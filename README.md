# Amazon-Sales-Analysis
Sales analysis project using Python, Pandas, and Matplotlib on Amazon Tech Trends dataset 202

## Tools Used
- Python
- Pandas
- Matplotlib

## Key Insights
- Computers category generated the highest revenue
- Accessories and Computers account for over 55% of total revenue
- Dataset contains 100 products across 7 categories
